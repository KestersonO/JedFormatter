# JedFormatter
Super simple MLA formatter, takes a history.com citation sheet and makes a Jed-approved citation

A **super** simple script (20~ Lines) that takes a 
HISTORY.COM citation copy/paste and 
formats it EXACTLY how Jed wants it. 

Just run it, paste in all the unedited
history.com info, then make a new line 
with just **END** on it, and it will spit 
out the MLA Format citation.
